# FMS_B737Sim
Documentation/config files for Flyhistorisk Museum Sola B737 Fly Simulator

Flysimulator: X-Plane 12 Pro (X-Plane)
 
737 Modell: Zibo (B737-800X (ZIBO mod))
 
Koblingen mellom flight controls/brytere/lys etc. er styrt av SimVimX System ( SimVimX
System )
 Noen av kontrollene er koblet via LeoBodnar kort (BU0836X joystick interface)
 
Display’s (PFD/ND/Engine/MFD etc) i front drives av zHSI (ZHSI)
 En plugin til X-Plane som kan kjøres på andre PC’er for å kunne vise
instrumentene på andre skjermer.
 
CDU’er er en «dum» boks som har en skjerm og knapper som snakker med flysimulatoren
via USB - PFP 3N (Panel of Flight Plan)
 
MCP, EFIS, AFT Electronics panel(Radios/NAV/XPNDR/Fire Panel etc.) er fra CPFlight
 Disse er noe av de som er gjenbruk av det som var i simulatoren fra før.
 
Throttle Quadrant: Denne er og gjenbruk, men har blitt kraftig modifisert for å få mer
funksjonalitet.
 
Prosjektorer: 3x Epson EB-L630U
 
PC’er:
 Simulator: Komplett-PC Epic Gaming i330 RGB, Windows 11 Home, Geforce
RTX 5080, Core Ultra 7 265KF, 32GB DDR5, 2TB SSD
 PFD/ND etc.: Lenovo ThinkCentre x2, som kjører zHSI

Lyd: Logitech Z906
