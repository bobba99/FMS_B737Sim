# FMS B737 Simulator
Dokumentasjon/config filer for Flyhistorisk Museum Sola B737 Fly Simulator

#### Flysimulator: X-Plane 12 Pro (X-Plane)

#### 737 Modell: Zibo (B737-800X (ZIBO mod))

#### Koblingen mellom flight controls/brytere/lys etc. er styrt av SimVimX System ( SimVimXSystem )
* Koblet med USB
* Noen av kontrollene er koblet via LeoBodnar kort (BU0836X joystick interface)

#### Display’s (PFD/ND/Engine/MFD etc) i front drives av zHSI (ZHSI)
* En plugin til X-Plane som kan kjøres på andre PC’er for å kunne vise
instrumentene på andre skjermer.

#### CDU’er er en «dum» boks som har en skjerm og knapper som snakker med flysimulatorenvia USB - PFP 3N (Panel of Flight Plan)
* Koblet med USB

#### MCP, EFIS, AFT Electronics panel(Radios/NAV/XPNDR/Fire Panel etc.) er fra CPFlight
* Koblet med USB
* Kjører en plugin
* Disse er noe av de som er gjenbruk av det som var i simulatoren fra før.

#### Throttle Quadrant: Denne er og gjenbruk, men har blitt kraftig modifisert for å få mer funksjonalitet.
* Koblet til en av BU0836X joystick interface
  
#### Prosjektorer: 3x Epson EB-L630U
* Koblet på nettverk og kan styres via app på PC

#### PC’er:
* Simulator: Komplett-PC Epic Gaming i330 RGB, Windows 11 Home, Geforce RTX 5080, Core Ultra 7 265KF, 32GB DDR5, 2TB SSD
* PFD/ND etc.: Lenovo ThinkCentre x2, som kjører zHSI, Linux Mint
* Instructor station: Lenovo Legion T730-28ICO Desktop - Type 90JF, Windows 10, Core i7-9700K, 16GB DDR4-2666, 1TB 7200rpm, 1TBNVMeOPAL2.0, NVIDIA®GeForce RTX™2080 SUPER™

#### Lyd: Logitech Z906
* Koblet med optisk kabel
* Kontroll enhet i cockpit
  
#### EFB: 2x iPAD
* For å kjøre Zibo EFB
* For å kjøre Navigraph Charts
